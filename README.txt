mod_whatsapp_button.zip Es modulo para instalar en joomla un simple boton de WhatsApp, solo hay que introducir el numero de teléfono en las opciones del modulo.
El archivo "codigo html boton WhatsApp" y el archivo "estilo css boton WhatsApp" son los codigos en html y css. El css lo tienes que colocar antes del /head y el código html antes del /body, en el codigo html tienes que sustituir "aquí va el numero de tu WhatsApp" incluidas las comillas por tu numero de teléfono.
